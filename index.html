World Clock

The world clock project is a web application coded in HTML, CSS, and JavaScript, utilizing a JavaScript framework. It displays the dates and times of different locations in various time zones. Users can select cities to view local times. The application updates in real-time, ensuring accurate and visually appealing time displays.

JavaScript Weather App

In my coding project, I utilized HTML, CSS, JavaScript, and OpenWeather API to create a weather application that provides real-time current forecasts for any city they choose. With a sleek interface, users can easily access weather conditions like humidity, wind speed, and feels-like temperature, making it a convenient tool for staying informed about the weather.

Virginia State Parks Redesign

For 2 weeks, I worked with a partner to improve the Virginia State Parks website. Our main focus was to make it easier for users to navigate and use the site. We did this by conducting usability testing and optimizing the website's design. We also added a cardinal logo that was inspired by Virginia's natural heritage. The final result is a visually appealing website that puts the user first and showcases the state's scenic beauty.
